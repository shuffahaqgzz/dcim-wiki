DCIM Block 5 — Web Dashboard

Reference Architecture • Vue 3 • API Gateway • RBAC/SSO • NOC • SOC • Facilities • CMDB Explorer • SLA/KPI • Logs • Task Board

7 Dashboard Views

  • • NOC — Real-time infrastructure health, alerts
  • • SOC — Security events, incidents, compliance
  • • Facilities — Power, cooling, environment
  • • CMDB Explorer — Interactive topology graph
  • • SLA/KPI — Compliance, trends, MTTR/MTBF
  • • Log Viewer — Centralized search + export
  • • Task Board — Kanban with SLA countdown

API Gateway Features

  • • JWT validation + SSO/OIDC
  • • Rate limiting (100/min read, 20/min write)
  • • RBAC (6 roles, resource-level)
  • • Response cache (5s-60s TTL)
  • • Circuit breaker per service
  • • CORS + CSP headers

Frontend Stack

  • • Vue 3 + Composition API
  • • Pinia state management
  • • ECharts (charts) + D3.js (topology)
  • • Tailwind CSS (dark theme)
  • • Vite (build) + TypeScript
  • • WebSocket (real-time push)

CMDB Explorer

  • • Interactive D3.js topology graph
  • • Node styling by type + status color
  • • Zoom/Pan/Drag/Click
  • • Depth-limited BFS traversal
  • • Impact analysis mode
  • • CI search + filter panel

Security

  • • SSO: Keycloak / Auth0 / Azure AD
  • • JWT token + silent refresh
  • • 6 RBAC roles
  • • Vue Router guards
  • • XSS/CSRF/CSP protection
  • • npm audit in CI

Responsive + Performance

  • • 3 breakpoints: mobile/tablet/desktop
  • • Code splitting (lazy load views)
  • • Virtual scrolling (log tables)
  • • Debounced search (300ms)
  • • Lighthouse target: > 80
  • • Bundle < 500KB gzipped
