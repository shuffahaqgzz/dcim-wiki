1. SIEM Core (LME Stack)

Wazuh Manager (3-node cluster) → Elasticsearch 8.x (hot/warm/cold) → Kibana dashboards. NiFi for data flow processing. 10+ detection rules mapped to MITRE ATT&CK. Log collection on port 1514, API on 55000.

2. Message Broker

Apache Kafka 3.x with 5 topics: events, alerts, cases, actions, enrichment. KRaft mode (no ZooKeeper). 3-broker cluster with replication factor 3. Central event bus for all SIEM/SOAR traffic.

3. SOAR Platform

TraceCat with built-in Temporal workflow engine. DFIR-IRIS case management with evidence tracking. Gemma/Unsloth local LLM for AI analysis. 8 standard YAML playbooks, OT-safe.

4. Security & Auth

Authentik OIDC/SSO on :9443. 5 RBAC roles (Admin, SOC L1/L2/L3, Auditor). TLS 1.2+ with mTLS. Full audit trail across all services.

5. Monitoring

Prometheus (:9090) + Grafana (:3000). Alert rules covering Kafka lag, ES cluster health, Wazuh agent status. SOC metrics dashboard for incident response KPIs.

6. Enrichment

AbuseIPDB for IP reputation. VirusTotal for hash/file scanning. iTop CMDB for asset context. GeoIP for location intelligence. All integrated via TraceCat playbooks.
